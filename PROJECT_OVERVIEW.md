---

## 📋 项目信息

### 项目名称
**自动邮件回复系统 (Automatic Email Reply System)**

### 项目目标
开发一个部署在学校官网上的自动邮件回复系统，通过邮件分类与 AI 自动回复，帮助教授快速处理学生邮件，节约时间。

---

## 🎯 核心功能

### 1. 邮件分类
- 按主题分类：
  - 学术问题（课程相关、作业、考试）
  - 行政事务（请假、证明、事务咨询）
  - 重复/低优先级邮件
- 自动识别重复问题并贴上标签，降低优先级

### 2. 自动回复
- 对低优先级或重复邮件进行自动回复
- 接入 LLM API（如 OpenAI/Azure OpenAI），生成简洁、合适的回复
- 回复可包含 FAQ 链接或固定说明

### 3. 管理面板
- 教授可查看分类结果与自动回复记录
- 提供人工介入选项（教授可选择是否手动回复）
- 可配置自动回复范围与模板

---

## 🏗️ 技术架构

### 前端
- HTML/CSS/JavaScript
- 嵌入学校官网

### 后端
- Python (Flask/FastAPI)
- 负责邮件处理与 API 调用

### 数据库
- MySQL/PostgreSQL
- 用于存储邮件、分类标签、回复记录

### AI 模块
- 文本分类模型
- LLM API 集成

### 安全
- SSL 加密
- 权限控制
- 保障邮件隐私

---

## 🔄 工作流程

1. 系统接收邮件
2. 分类处理（规则 + AI 模型）
3. 判断优先级（重复/低优先级）
4. 自动生成回复（调用 LLM API）
5. 教授可审核或修改（可选）
6. 系统发送邮件

---

## ✨ 项目亮点

- **智能化** - 结合规则与 AI 分类，提升准确率
- **节省时间** - 教授无需处理重复性问题
- **可扩展性** - 未来可支持多语言、个性化回复
- **透明性** - 教授可随时查看和调整自动回复
