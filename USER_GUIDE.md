# User Guide — Automatic Email Reply System

> **Important:** For presentation and grading purposes, please use the **Offline Demo Mode** (no API keys or internet required).

---

## Option 1: Offline Demo Mode (Recommended)

This mode runs entirely on your machine with no external services needed. It uses a local SQLite database and template-based replies instead of calling the DeepSeek API.

### Prerequisites

- Python 3.10 or higher

### Steps

```bash
git clone https://github.com/3031445204zelsy17-lang/COMP2090SEF_group_project_GRP_69.git
cd COMP2090SEF_group_project_GRP_69/task1
pip install -r requirements.txt
python run_demo.py
```

Then open **http://localhost:8000** in your browser.

### Demo Login Credentials

| Role | Email | Password |
|------|-------|----------|
| **Professor** | professor@hkmu.edu.hk | demo123 |
| Student | student@hkmu.edu.hk | demo123 |

### What You Can Do in Demo Mode

1. **Inbox** — View pre-loaded student emails, classified by category (Academic, Administrative, FAQ) and sorted by priority
2. **Auto-Reply** — Click an email to see the AI-generated reply. Use the Chat Edit box to modify it (e.g., type "reply in English" or "make it shorter"), then approve and send
3. **Knowledge Base** — Navigate to the Sources tab to view, add, edit, or delete course materials and FAQs
4. **Statistics** — Check the Stats tab for email volume, reply rates, and other metrics

## Option 2: Full Mode (API Required)

Full mode connects to DeepSeek for AI-powered replies and Supabase for cloud database storage.

### Prerequisites

- Python 3.10 or higher
- A DeepSeek API key (https://platform.deepseek.com)
- (Optional) A Supabase project for cloud storage

### Steps

```bash
git clone https://github.com/3031445204zelsy17-lang/COMP2090SEF_group_project_GRP_69.git
cd COMP2090SEF_group_project_GRP_69/task1
pip install -r requirements.txt
cp .env.example .env
# Edit .env and fill in your API key and other settings
cd src
python main.py
```

Then open **http://localhost:8000** in your browser.

Register a professor account through the login page, then use the system as described in Demo Mode. In full mode, replies are generated by DeepSeek and are context-aware based on your knowledge base entries.
